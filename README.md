#MBSet
=====

##Requirements:
=====
This application requires a CUDA compatible NVIDIA graphics card and CUDA 4.0 installed

##Compile:
=====
1. Edit the Makefile and change the path to your CUDA installation
2. Run:
```
make
```

##Usage:
=====

Run (exports CUDA library to path on run):
```
sh runMBSet.sh
```
